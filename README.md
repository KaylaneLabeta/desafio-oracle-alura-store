# 📊 Análise de Desempenho de Lojas

## 📌 Sobre o Projeto

Este projeto consiste em uma análise exploratória de dados com o objetivo de avaliar o desempenho de quatro lojas e identificar qual delas apresenta menor competitividade estratégica.

A análise foi desenvolvida utilizando Python no Google Colab, aplicando técnicas de manipulação e visualização de dados para gerar insights que auxiliem na tomada de decisão.

O principal desafio do projeto foi transformar dados brutos de vendas em informações estratégicas capazes de embasar uma recomendação gerencial.

---

## 🎯 Objetivo

O objetivo do projeto foi responder à seguinte questão:

> **Qual loja deve ser vendida com base nos indicadores de desempenho analisados?**

Para isso, foram considerados fatores financeiros, operacionais e de satisfação do cliente.

---

## 📊 Análises Realizadas

Durante o desenvolvimento do projeto, foram realizadas as seguintes análises:

- 💰 Cálculo do faturamento total por loja  
- 📦 Identificação das categorias mais e menos vendidas  
- 🛍 Análise dos produtos com maior e menor volume de vendas  
- ⭐ Cálculo da média de avaliação dos clientes por loja  
- 🚚 Cálculo do custo médio de frete por loja  
- 📈 Geração de gráficos comparativos para visualização dos resultados  
- 📝 Elaboração de relatório estratégico com recomendação final  

---

## 🛠 Ferramentas e Tecnologias Utilizadas

- **Python** – Linguagem principal para análise  
- **Pandas** – Manipulação e tratamento dos dados  
- **Matplotlib** – Criação de gráficos  
- **Seaborn** – Visualização estatística  
- **Google Colab** – Ambiente de desenvolvimento  

---

## 📈 Principais Insights

A análise permitiu identificar diferenças importantes entre as lojas, considerando:

- Desempenho financeiro  
- Eficiência operacional  
- Satisfação do cliente  
- Competitividade geral  

Com base no conjunto dos indicadores analisados, foi possível recomendar a venda da loja com menor desempenho estratégico.

---

## 📌 Conclusão

Este projeto demonstra a aplicação prática de análise de dados para suporte à tomada de decisão empresarial. Através da exploração, organização e interpretação dos dados, foi possível transformar números em uma recomendação estratégica clara e fundamentada.
